## Como jogar
=== Comandos do player vermelho <br>
  W -> Pra cima <br>
  S -> Pra baixo <br>
  D -> Atira <br>
  R -> Recarregar <br>

=== Comandos do player azul <br>
  Seta pra cima -> Pra cima <br>
  Seta pra baixo -> Pra baixo <br>
  Seta pra esquerda -> Atira <br>
  Seta pra direita -> Recarrega <br>
